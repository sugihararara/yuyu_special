## 気合いの入ったパンチorクリーンヒット発生率計算式(16進数)

パンチ力UPor霊撃力UPorドーピングの「キレた～」効果=CD
  
瀕死補正値=EF
  
AB+CD+EF+1=GH
  
GH/100=IJ
