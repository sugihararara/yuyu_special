## 乱数補正計算式(16進数)(未確認要検証)

[闘気補正](https://w.atwiki.jp//w.atwiki.jp/yuyuz/pages/19.html "闘気補正 (1951d)")成功率or回避率=ABCD
  
乱数補正値×CD=EFGH(下2桁は使用しない)
  
乱数補正値×AB=IJKL
  
EF+KL=MN(繰り上がり桁は使用しない)
  
NMとIJを結合=MNIJ
  
乱数補正成功率or回避率=MNIJ

## 乱数補正値

どちらかが非攻撃系時、返し成立時、飛び同士の相殺判定時、後手側の乱数が特殊

#### 16進数

|  |  |
| --- | --- |
| 通常 | C0~FF |
| 特殊 | 80~FF |

#### 10進数

|  |  |
| --- | --- |
| 通常 | 192~255 |
| 特殊 | 128~255 |
