## まとめサイト作成支援ツールについて

@wikiには[まとめサイト作成を支援するツール](https://w.atwiki.jp//atwiki.jp/matome/)があります。
  
また、

```
#matome_list
```

と入力することで、注目の掲示板が一覧表示されます。

  

利用例）#matome\_listと入力すると下記のように表示されます

#matome\_list

タグ：

+ タグ編集

* タグ：

キャンセル
